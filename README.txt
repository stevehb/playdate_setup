---=== PLAYDATE STARTER ===---

Playdate SDK is installed at 
C:\codes\PlaydateSDK


The Playdate ARM compiler toolchain can be found here:
C:\ProgramData\chocolatey\lib\gcc-arm-embedded\tools\gcc-arm-none-eabi-10.3-2021.10\bin

Run `make simulator` to build the project.
Run `make run_simulator` to run in the simulator.
